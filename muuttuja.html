6. Muuttuja

Esimerkki 1

Piirretään piirtoikkunaan mahdollisimman suuri ympyrä. Ympyrän halkaisijan pituus riippuu siitä, kumpi on suurempi: ikkunan leveys vai korkeus. Ensimmäisessä esimerkissä leveys on suurempi, joten ympyrän halkaisija on ikkunan korkeus. 

function setup() {
 createCanvas(300, 200);
 background("lightblue");
 if (width > height)
   ellipse(width / 2, height / 2, height, height);
 else
   ellipse(width / 2, height / 2, width, width);
}

Kun ikkunan mittasuhteet muuttuvat, ympyrän halkaisija on ikkunan leveys. 

function setup() {
 createCanvas(300, 400);
 background("lightblue");
 if (width > height)
   ellipse(width / 2, height / 2, height, height);
 else
   ellipse(width / 2, height / 2, width, width);
}

Huomataan, että ohjelmaan on kirjoitettu funktion ellipse() kutsu kahdesti, vaikka vain ympyrän halkaisija muuttuu. Voisiko tämän hoitaa fiksummin?

Sama esimerkki muuttujan avulla

Tietokoneohjelmassa muuttuja on nimetty muistipaikka, johon voidaan tallentaa mm. lukuja. Muuttujan arvo voi muuttua ohjelman suorituksen aikana.

Ohjelmoidaan yllä olevan esimerkin mukainen ohjelma käyttämällä muuttujaa, jonka nimeksi annetaan halkaisija. 

function setup() {
  createCanvas(400, 300);
  background("lightblue");

  let halkaisija; // määritellään muuttuja nimeltä halkaisija, ei arvoa
  if (width > height)
    halkaisija = height; // halkaisijan arvoksi tulee ikkunan korkeus
  else
    halkaisija = width; // halkaisijan arvoksi tulee ikkunan leveys

  ellipse(width / 2.0, height / 2.0, halkaisija, halkaisija);
}

Huomaa, että

  • muuttuujaa määritellessä sen nimen edessä lukee let
  • muuttujalle ei tarvitse antaa arvoa määrittelevällä rivillä
  • muuttujalle kuitenkin voidaan antaa heti arvo, esim. let halkaisija = 10
  • muuttujalle asetetaan arvo yhtäsuuruusmerkin = avulla.

Monet ohjelmoijat kirjoittavat muuttujien nimet pienillä kirjaimilla erotellakseen ne vakioista.

Esimerkki 2

Piirretään ikkunaan palkki, joka peittää P prosenttia ikkunasta, missä P on vakio. Jos vakion P arvo on alle 90, palkki on sininen. Muussa tapauksessa palkki on vihreä.

Ensimmäisessä tapauksessa vakion P arvo on alle 90. 

function setup() {
  createCanvas(600, 100);
  background("lightlblue");
  const P = 65;

  const LEVEYS = P / 100 * width; // palkin leveys

  colorMode(HSB, 100);
  noStroke();

  let varisavy; // palkin värin sävy
  if (P < 90)
    varisavy = 65; // sininen
  else
    varisavy = 30; // vihreä

  fill(varisavy, 70, 100);
  rect(0, 0, LEVEYS, height);
}

Kun vakion P arvo kasvaa, palkin väri muuttuu vihreäksi. 

function setup() {
 createCanvas(600, 100);
 background("lightblue");
 const P = 95;

 const LEVEYS = P / 100 * width; // palkin leveys

 colorMode (HSB, 100);
 noStroke ();

 let varisavy; // palkin värin sävy
 if (P < 90)
   varisavy = 65; // sininen
 else
   varisavy = 30; // vihreä

 fill(varisavy, 70, 100);
 rect(0, 0, LEVEYS, height);
}

Esimerkki 3

Piirretään mahdollisimman suuri ellipsi, jonka keskipisteen \(x\)-koordinaatti on vakio X. Korkeussuunnassa ellipsin keskipiste on ikkunan puolivälissä.

Ensimmäisessä tilanteessa keskipisteen etäisyys vasempaan reunaan määrittää ellipsin leveyden. 

function setup() {
  createCanvas(400, 300);
  background("lightblue");
  const X = 100;

  let leveys;
  // keskipisteen etäisyydet ikkunan oikeasta ja vasemmasta reunasta
  const ETAISYYS_OIKEA = width - X;
  const ETAISYYS_VASEN = X;

  if (ETAISYYS_VASEN < ETAISYYS_OIKEA)
    leveys = 2 * ETAISYYS_VASEN;
  else
    leveys = 2 * ETAISYYS_OIKEA;

  ellipse(X, height / 2.0, leveys, height);
}

Toisessa tilanteessa vakion X arvo on suurempi, ja ellipsin leveys määräytyy keskipisteen etäisyydestä ikkunan oikeaan reunaan. 

function setup() {
  createCanvas(500, 100);
  background("lightblue");
  const X = 440;

  // keskipisteen etäisyydet ikkunan oikeasta ja vasemmasta reunasta
  const ETAISYYS_OIKEA = width - X;
  const ETAISYYS_VASEN = X;

  let leveys;
  if (ETAISYYS_VASEN < ETAISYYS_OIKEA)
    leveys = 2 * ETAISYYS_VASEN;
  else
    leveys = 2 * ETAISYYS_OIKEA;

  ellipse(X, height / 2.0, leveys, height);
}

Alla on vaihtoehtoinen, lyhyempi ratkaisu. Etäisyys ikkunan reunaan voidaan päätellä myös suoraan siitä, sijaitseeko keskipiste ikkunan vasemmassa vai oikeassa puoliskossa. 

function setup() {
 createCanvas(200, 300);
 background("lightblue");
 const X = 150;

 let etaisyys; // keskipisteen etäisyys ikkunan reunaan
 if (X < width / 2.0)
   etaisyys = X; // keskipiste ikkunan vasemmassa puoliskossa
 else
   etaisyys = width - X;

 ellipse(X, height / 2.0, 2 * etaisyys, height);
}

Tehtäviä

Ratkaise seuraavat tehtävät muuttujien avulla.

  1. Kirjoita ohjelma, joka piirtää \((400 \times 400)\)-kokoisen piirtoikkunan keskelle ympyrän. Määrittele ohjelmassasi muuttuja ympyrän halkaisijalle. Halkaisija on 100, jos vakion K arvo on alle 100. Muussa tapauksessa halkaisija on 200. Testaa ohjelmasi toimivuus muuttamalla vakion K arvoa.
  2. Kirjoita ohjelma, joka piirtää \((400 \times 400)\)-kokoisen piirtoikkunan vasempaan ylänurkkaan ympyrän. Määrittele ohjelmassasi muuttuja ympyrän halkaisijalle. Halkaisija on 100, jos vakion K arvo on alle 100. Muussa tapauksessa halkaisija on 200. Testaa ohjelmasi toimivuus muuttamalla vakion K arvoa.
  3. Kirjoita ohjelma, joka piirtää piirtoikkunan keskelle mahdollisimman suuren valkoisen neliön. Testaa ohjelmasi toimivuus muuttamalla piirtoikkunan kokoa.
  4. Piirrä piirtoikkunan keskelle mahdollisimman suuri ympyrä, joka toteuttaa seuraavan ehdon: ympyrän ja piirtoikkunan väliin jää joka suunnassa tilaa vähintään ympyrän säteen verran.
  5. Kirjoita ohjelma, jossa määritellään vakiot X1 ja X2, joiden molempien arvo on vähintään 0 ja korkeintaan 300. Ohjelma piirtää \((400 \times 400)\)-kokoiseen piirtoikkunan yläreunaan \((100\times 100)\)-kokoisen neliön, jonka vasemman yläkulman \(x\)-koordinaatti on pienempi vakioista X1 ja X2 (kumpi tahansa vakioista, jos vakiot ovat yhtä suuret). Testaa ohjelmasi toimivuus muuttamalla vakioiden arvoja.

  6. a) Jaetaan neliön muotoinen piirtoikkuna kahteen osaan alla olevan kuvan mukaisesti.

    Mikä ehto pätee niiden pisteiden \(x\)- ja \(y\)-koordinaateille, jotka sijaitsevat ikkunan punaisessa osassa? b) Kirjoita ohjelma, jossa määritellään vakiot X ja Y, joiden molempien arvo on vähintään 100 ja korkeintaan 300. Ohjelma piirtää \((400\times 400)\)-kokoiseen piirtoikkunaan ympyrän, jonka keskipiste on kohdassa (X, Y) ja jonka halkaisija on 10. Ympyrä on punainen, jos sen keskipiste sijaitsee ikkunan “punaisessa” osassa. Muussa tapauksessa ympyrä on sininen.

  7. Kirjoita ohjelma, joka piirtää neliön muotoiseen piirtoikkunaan mahdollisimman suuren ympyrän, joka täyttää seuraavat ehdot.

    • Ohjelman vakio X määrää ympyrän keskipisteen \(x\)-koordinaatin.
    • Ympyrän keskipiste on piirtoikkunan lävistäjällä, jonka toinen pää on origossa.

    Alla on ohjelman piirtämä kuva, kun ikkunan koko on \((400\times 400)\) ja vakion X arvo on 370.
