7. while-silmukka

while-silmukka

Lauselohkot { }

Lauselohko yhdistää toisiinsa useita lausekkeita. Lauselohkon avulla esimerkiksi if:in toteutuessa voidaan suorittaa useita komentoja.

Lauselohko alkaa {-merkillä ja päättyy }-merkkiin. if-else-rakenteessa lauselohkot näyttävät seuraavilta. 

if (ehto)
{
  lauseke;
  toinen lauseke;
  ...
  viimeinen lauseke;
}
else
{
  lauseke;
  toinen lauseke;
  ...
  viimeinen lauseke;
}

Esimerkki lauselohkon käytöstä, missä piirrettävän ympyrän koko ja taustaväri riippuvat siitä, onko piirtoikkunan leveys suurempi kuin korkeus: 

function setup() {
  createCanvas(200, 300);
  noStroke(); // ei piirretä ääriviivoja

  let halkaisija; // otetaan käyttöön muuttuja: halkaisija
  if (width > height)
  {
    background("black"); // musta tausta
    halkaisija = 100;
  }
  else
  {
    background("blue"); // sininen tausta
    halkaisija = 50;
  }

  ellipse (width / 2, height / 2, halkaisija, halkaisija);
}

Piirtämista while-silmukalla

Piirretään alla oleva kuva, jossa on \(20\times 20\)-kokoisia neliöitä piirtoikkunan lävistäjällä.

Sen sijaan, että joutuisimme kirjoittamaan yhden koodirivin kutakin neliötä kohden, ohjelma voidaan kirjoittaa while-kontrollirakenteen avulla. 

function setup(){
  createCanvas(400, 400);
  background("lightblue");

  const SIVU = 20;
  noStroke(); // ei piirretä neliöiden ääriviivoja

  let k = 0; // neliön yläkulman x- ja y-koordinaatti

  while (k < width) // niin kauan kuin ollaan ikkunan sisällä...
  {
   // ... suoritetaan tämä lauselohko
   rect (k, k, SIVU, SIVU); // piirretään neliö
   k = k + SIVU; // kasvatetaan yläkulman koordinaattia
  }
}

while-rakenne

while-rakenteessa on

  1. ehto, joka määrää kuinka kauan lauseke tai lausekelohko { } suoritetaan
  2. lauseke tai lausekelohko, joka suoritetaan niin kauan kuin ehto on tosi
while (ehto)
  lauseke tai lohko, joka suoritetaan niin kauan kuin ehto tosi;

Huomaa, että

  • ehto kirjoitetaan sulkujen sisään
  • while-rivin lopussa ei ole puolipistettä.

while-rakennetta kutsutaan yleisemmin while-silmukaksi tai while-loopiksi.

Esimerkki 1

Piirretään \(400\times 400\)-kokoiseen ikkunaan alla olevan kaltainen kuva. Kunkin viivan kuvan alareunassa olevan päätepisteen \(x\)-koordinaatti määrää viivan värisävyn siten, että värisävy on \(\displaystyle\frac{x}{4}.\) Kuvan alareunan vierekkäisten päätepisteiden väli on 5 pikseliä. 

function setup() {
  createCanvas(400, 400);
  colorMode(HSB, 100);
  background (0); // musta tausta

  let x = 0; // alareunassa olevan päätepisteen x-koordinaatti
  while (x < width) // niin kauan kun piste on kuvan sisällä
  {
    stroke(x / 4, 100, 100); // asetetaan väri
    line(0, 0, x, height); // piirretään viiva
    x = x + 5;
  }
}

Esimerkki 2

Piirretään alla oleva kuva, jossa neliön muotoiseen piirtoikkunaan piirretään ruudukko. Ruudukossa vierekkäisten yhdensuuntaisten viivojen välinen etäisyys on vakio L.

Yllä olevassa esimerkissa piirtoikkunan koko on \(401\times 401\) ja vakion L arvo 10. Tällöin viimeinen pystyviiva piirtyy ruudukon oikeaan reunaan: ensimmäinen pystyviiva piirretään 1. pikselin kohdalle, toinen pystyviiva 11. pikselin kohdalle jne., jolloin viimeinen kuvaan piirrettävä pystyviiva tulee 401. pikselin kohdalle.

Kirjoitetaan kaksi erilaista ratkaisua. Ensimmäisessä piirretään ensin pystyviivat yhdessä while-silmukassa ja sen jälkeen vaakaviivat toisessa silmukassa. 

function setup() {
  function setup() {
    createCanvas(401, 401);
    const L = 10; // viivojen välinen etäisyys

    let x = 0; // pystysuoran viivan x-koordinaatti
    while (x < width)
    {
      line(x, 0, x, height);
      x = x + L;
    }

    let y = 0; // vaakasuoran viivan y-koordinaatti
    while (y < height)
    {
      line(0, y, width, y);
      y = y + L;
    }
  }
}

Toisessa ratkaisussa riittää yksi silmukka, jonka sisällä piirretään yksi vaakasuora ja yksi pystysuora viiva. 

function setup() {
  createCanvas(401, 401);
  const L = 10; // viivojen välinen etäisyys

  let p = 0; // viivan toinen koordinaatti
  while (p < width)
  {
    line(p, 0, p, height); // pystysuora viiva
    line(0, p, width, p); // vaakasuora viiva
    p = p + L;
  }
}

Esimerkki 3

Piirretään alla oleva murtoviiva piirtoikkunaan, jonka koko on \(600\times 400\). Murtoviivan “ensimmäinen piste” on piirtoikkunan vasemmassa alakulmassa. Jos viivan tämänhetkinen kärkipiste on kohdassa \((x, y)\), niin seuraava kärkipiste on kohdassa \(\displaystyle\left(x + 100, \frac{y}{2}\right)\).

Alla olevassa ohjelmassa muuttujat x ja y sisältävät tiedon piirrettävän viivan kärkipisteestä. Kullakin kierroksella while-silmukan sisällä lasketaan seuraavan kärkipisteen koordinaatit muuttujiin xSeuraava ja ySeuraava, piirretään viiva ja päivitetään kärkipisteen koordinaatit. 

function setup() {
  createCanvas(600, 400);
  colorMode(HSB, 100);
  background (0);
  stroke(100);

  let x = 0, y = height;

  while (x < width)
  {
   let xSeuraava = x + 100;
   let ySeuraava = y / 2;

   line(x, y, xSeuraava, ySeuraava);

   x = xSeuraava;
   y = ySeuraava;
  }
}

Huomaa, että ohjelmakoodissa muuttujat xSeuraava ja ySeuraava määritellään lauselohkon sisällä. Ne voitaisiin määritellä myös lauselohkon ulkopuolella, mutta niitä ei tarvita siellä. Tähän asiaan palataan kappaleessa, jossa määritellään funktioita.
