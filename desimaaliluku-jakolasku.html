Desimaaliluku ja jakolasku

Desimaalierotin

JavaScript-ohjelmointikielessä (ja yleensäkin ohjelmointikielissä) desimaalierotin on piste. Seuraava esimerkki piirtää kohtaan \((100, 100)\) suorakulmion, jonka korkeus on leveyden ja luvun 0,3 tulo. (Toisin sanoen korkeus on 30 % leveydestä.) 

function setup() {
  createCanvas(400, 400);
  background("lightblue");
  const LEVEYS = 150;
  rect(100, 100, LEVEYS, 0.3 * LEVEYS);
}

Jakolasku

Jakolaskuoperaattori JavaScript-kielessä on kauttaviiva /. Alla olevassa esimerkissä piirretään neliö ja sen kanssa kulmittain suorakaide, joka leveys on puolet ja korkeus kolmasosa neliön sivun pituudesta. 

function setup() {
  createCanvas(400, 400);
  background("lightblue");
  const SIVU = 150;
  rect(0, 0, SIVU, SIVU);
  rect(SIVU, SIVU, SIVU / 2, SIVU / 3);
}

Kokonaislukujen jakolasku antaa kokonaisluvun

Jakolaskuun liittyy suorastaan vaarallinen yksityiskohta.

Kahden kokonaisluvun jakolasku antaa tulokseksi kokonaisluvun. Tulos on jakolaskun kokonaislukuosa

Alla olevaan taulukkoon on kirjattu muutamia jakolaskuja sekä niiden antama tulos.

jakolasku tulos
10 / 2 5
8 / 5 1.6
1 / 5 0.2

Jos jakolasku ei mene tasan kokonailukujen kannalta, tulee lukuun mukaan pisteellä erotettu desimaaliosa.

Joskus jakolaskun tulokseen ei haluta desimaaliosaa lainkaan. Tämä onnistuu laittamalla jakolasku funktion int() sisällä oleviin sulkeisiin. Funktion nimi tulee englannin kielen kokonaislukua tarkoittavasta sanasta integer.

jakolasku tulos
int(10 / 2) 5
int(8 / 5) 1
int(1 / 5) 0

Sisäiset “vakiot” width ja height ja skaalautuvat kuvat

p5.js-ohjelma määrittelee automaattisesti “kokonaislukuvakiot” width ja height, joiden arvot ovat piirtoikkunan leveys ja korkeus. Periaatteessa ikkunan koko voi muuttua ohjelman suorituksen aikana, joten luvut eivät tarkalleen ottaen ole vakioita, mutta meidän ohjelmissamme ne eivät muutu.

Seuraavassa esimerkissä piirretään neliön muotoiseen piirtoikkunaan mahdollisimman suuri ympyrä. Laskutoimitus width / 2 antaa puolet neliön sivun pituudesta eli ympyrän säteen. Ympyrä piirretään käyttämällä funktiota ellipse(), jolla voidaan piirtää eri kokoisia ellipsejä. Funktiolle annetaan neljä lukua. Kaksi ensimmäistä lukua määräävät ellipsin keskipisteen. Seuraava luku määrää ellipsin leveyden ja viimeinen ellipsin korkeuden. Jos leveys ja korkeus ovat samat, saadaan ympyrä. 

function setup() {
  createCanvas(400, 400);
  background("lightblue");
  // ympyrän säde
  const SADE = width / 2;
  ellipse(SADE, SADE, width, width);
}

Piirretty kuva skaalautuu automaattisesti piirtoikkunan koon muuttuessa, eli ympyrä täyttää neliön muotoisen ikkunan, vaikka ikkuna olisi suurempi tai pienempi. 

function setup() {
  createCanvas(100, 100);
  background("lightblue");
  // ympyrän säde
  const SADE = width / 2;
  ellipse(SADE, SADE, width, width);
}

Esimerkki 1

Suomen, Ruotsin ja Tanskan lipuissa on jonkin värinen risti toisen värisellä pohjalla. Värien lisäksi lipuissa vaihtelevat mittasuhteet. Esimerkiksi Ruotsin lippu jakautuu eri osiin vasemmasta yläkulmasta katsoen pystysuunnassa suhteessa \(4 : 2 : 4\) ja vaakasuunnassa suhteessa \(5 : 2 : 9\). Alla oleva kuva havainnollistaa näitä mittoja.

Alla olevalla ohjelmalla voidaan piirtää neliön muotoiseen ikkunaan vakioita vaihtelemalla Ruotsin, Tanskan tai Suomen lippu. Vakioina annetaan mittasuhteet sekä värien tiedot. Lippu piirretään koko piirtoikkunan levyisenä. Ensimmäisenä lasketaan, mikä on tällöin lipun mitoissa olevan yhden “osan” suuruus. Jos leveys on vaikkapa 600, ja kyseessä on Ruotsin lippu, niin yhden osan suuruus on tällöin \[ \frac{600}{5 + 2 + 9} = 37,5. \] 

function setup() {
   createCanvas(600, 600);
   // suhteet vaakasuunnassa
   const VASEN = 5;
   const KESKI_LEVEYS = 2;
   const OIKEA = 9;

   // suhteet pystysuunnassa
   const YLA = 4;
   const KESKI_PYSTY = 2;
   const ALA = 4;

   // taustaväri
   const TAUSTA_SAVY = 58;
   const TAUSTA_KYLLAISYYS = 100;
   const TAUSTA_KIRKKAUS = 26;

   // ristin väri
   const RISTI_SAVY = 13;
   const RISTI_KYLLAISYYS = 100;
   const RISTI_KIRKKAUS = 100;

   // yhden osan pituus kuvassa: kuvan leveys jaettuna vaakasuunnan
   // osien summalla
   const OSA = width / (VASEN + KESKI_LEVEYS + OIKEA);

   // lipun korkeus
   const KORKEUS = (YLA + KESKI_PYSTY + ALA) * OSA;

   colorMode(HSB, 100);
   noStroke();

   // lipun tausta
   fill(TAUSTA_SAVY, TAUSTA_KYLLAISYYS, TAUSTA_KIRKKAUS);
   rect(0, 0, width, KORKEUS);

   // risti
   fill(RISTI_SAVY, RISTI_KYLLAISYYS, RISTI_KIRKKAUS);
   rect(VASEN * OSA, 0, KESKI_LEVEYS * OSA, KORKEUS); // pystypalkki
   rect(0, YLA * OSA, width, KESKI_PYSTY * OSA); // vaakapalkki
  }

Muuttamalla kuvan kokoa sekä vakioita, jotka säätelevät mittasuhteita ja värejä saadaan pienempi Tanskan lippu. Tanskan lipussa vaakasuunta jakautuu suhteissa \(12 : 4 : 21\) ja pystysuunta vastaavissa mitoissa suhteessa \(12 : 4: 12.\) 

function setup() {
 createCanvas(350, 350);
 // suhteet vaakasuunnassa
 const VASEN = 12;
 const KESKI_LEVEYS = 4;
 const OIKEA = 21;

 // suhteet pystysuunnassa
 const YLA = 12;
 const KESKI_PYSTY = 4;
 const ALA = 12;

 // taustaväri
 const TAUSTA_SAVY = 97;
 const TAUSTA_KYLLAISYYS = 92;
 const TAUSTA_KIRKKAUS = 78;

 // ristin väri
 const RISTI_SAVY = 0;
 const RISTI_KYLLAISYYS = 0;
 const RISTI_KIRKKAUS = 100;

 // yhden osan pituus kuvassa: kuvan leveys jaettuna vaakasuunnan
 // osien summalla
 const OSA = width / (VASEN + KESKI_LEVEYS + OIKEA);

 // lipun korkeus
 const KORKEUS = (YLA + KESKI_PYSTY + ALA) * OSA;

 colorMode(HSB, 100);
 noStroke();

 // lipun tausta
 fill(TAUSTA_SAVY, TAUSTA_KYLLAISYYS, TAUSTA_KIRKKAUS);
 rect(0, 0, width, KORKEUS);

 // risti
 fill(RISTI_SAVY, RISTI_KYLLAISYYS, RISTI_KIRKKAUS);
 rect(VASEN * OSA, 0, KESKI_LEVEYS * OSA, KORKEUS); // pystypalkki
 rect(0, YLA * OSA, width, KESKI_PYSTY * OSA); // vaakapalkki
}

    function setup() {
  createCanvas(500, 200);
  background("lightblue");
  const A = 1;
  const B = 3;
  const C = 3;
  const D = 2;
  noStroke();

  // yhteisen kulmapisteen koordinaatit
  const X = A * width / (A + B);
  const Y = C * height / (C + D);

  rect(0, 0, X, Y);
  rect(X, Y, width - X, height - Y);
}
